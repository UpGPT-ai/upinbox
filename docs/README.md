## Billing

All billing is at [upgpt.ai](https://upgpt.ai). UpInbox is a capability in your UpGPT subscription.

The 'email' capability unlocks hosted UpInbox at mail.upinbox.ai and the Inbox tab in UpLink mobile.
The 'mcp' capability unlocks the MCP server.
The 'byok' capability unlocks BYOK AI configuration.

Self-hosters get full UpInbox web + PWA features without an UpGPT subscription. Native UpLink mobile Inbox still requires UpGPT.
